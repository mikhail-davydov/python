# Django Framework — Приложение "Опросы"

## Описание

Учебное Django-приложение "Опросы" (polls), созданное на основе [официального Django Tutorial](https://docs.djangoproject.com/en/5.2/intro/). Демонстрирует основные концепции и компоненты Django-фреймворка.

## Функциональность

- **Создание опросов** — администратор может создавать вопросы с вариантами ответов
- **Голосование** — пользователи могут выбирать варианты ответов
- **Результаты** — отображение процентов голосов для каждого варианта
- **Админ-панель** — встроенная Django-админка для управления данными

## Установка и запуск

### Требования

- Python 3.10+
- Django 5.x

### Установка зависимостей

```bash
cd django
pip install django
```

### Запуск сервера разработки

```bash
python manage.py runserver
```

Приложение будет доступно по адресу: http://127.0.0.1:8000/

### Создание миграций

```bash
python manage.py makemigrations
python manage.py migrate
```

### Создание суперпользователя

```bash
python manage.py createsuperuser
```

## Тестирование

Запуск встроенных тестов:

```bash
python manage.py test
```

## Используемые технологии

- Django 5.x — веб-фреймворк
- SQLite — встроенная база данных
- Bootstrap 5 — CSS-фреймворк для шаблонов
- pytest-django — для расширенного тестирования
