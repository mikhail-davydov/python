# Игра "Змейка" (Snake Game)

## Описание

Классическая аркадная игра "Змейка", реализованная на Python с использованием библиотеки `pygame`. Игра демонстрирует базовые принципы разработки игр, обработки пользовательского ввода и работы с графикой.

## Правила игры

1. Управляйте змейкой, которая движется по игровому полю
2. Собирайте еду (зелёные блоки), чтобы змейка росла
3. Избегайте столкновений со стенами и собственным хвостом
4. Наберите как можно больше очков

## Управление

| Клавиша | Действие |
|---------|----------|
| Стрелка вверх | Движение вверх |
| Стрелка вниз | Движение вниз |
| Стрелка влево | Движение влево |
| Стрелка вправо | Движение вправо |
| Space / Enter | Новая игра (после проигрыша) |
| ESC | Выход из игры |

## Технические детали

| Параметр | Значение |
|----------|----------|
| Библиотека | pygame |
| FPS | 15 |
| Размер клетки | 10x10 пикселей |
| Размер поля | 300x300 пикселей |

### Требования

- pygame


```bash
cd snake
```
